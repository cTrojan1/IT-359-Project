# IT-359-Project
Our project for this course is going to be an ethical keylogging software that will allow us to write keystrokes to a .txt file and read the user inputs.

## Full Project Idea
Our team is going to create a keylogging software that will allow us to read user inputs on a specefied machine.
# Key components
- TXT file to store key strokes
- Exportable code to a USB drive for portability
- Wireless transmission upon USB insertion
- Interval screenshots
- Type of input related to the keystrokes (passwords, usernames, etc.)
- Timestamps relating to the logged information
